扎心了

发表于 2019-04-05 | 分类于 破裂 | 次阅读

2017年6月30号，我给她发微信，说周末了我去找你吧，她直接扔过来一句，我回家了，我说你回家怎么不告诉我一声呢，她说跟你有什么关系，当时距离结婚还有120天左右，我很生气，因为我们俩都在郑州，你回家不应该告诉你的未婚夫一声吗，我随口回了一句那你随便
下午对方发来微信，说你回来吧，我把所有的都退给你，当时我正在上班没反应过来，问对方啥意思，对方说我们分手吧。
120天就要结婚的未婚妻跟我说出这样的话，我也是年轻气盛，气头上随口回了一句，分就分，对方说那彩礼怎么办，我回了句哪来的还回哪里去，当时我纯粹是气话，没想到对方当真的，于是就成了真的。这是我长这么大唯一让我悔恨终生的一句话，也许自己不那么敏感，不那么小心眼，或者以成熟的心态对待这件事，可能就是另外的一个场景的。
随后就是我的认错，以及挽回过程。
7月1号晚上我给她打电话，她说的话总结如下：
我想要的是浪漫和爱情，你给不了
你日子过得太节俭了，你去找一个勤俭节约的媳妇去吧
你总是一个人忙，都顾不上我，你不珍惜我有人懂得珍惜
我说我以前没有意识的自己这么多的缺点，现在我知道错了，我改正重新来过好不好，对方说不可能了。
（一星期左右微信消息）我开始新恋爱了，希望你不要再来打扰我了，我不想有一段没有结果的恋爱。
当时我已经有点神经了
随后她朋友圈发了一句：再见青春，勿念勿扰！
7月10号左右，我登录对方微信，发现一个聊得特别亲密的男性朋友，气不过将之删除，随后对方电话打过来质问我，为什么要删除其好友，我说你是我女朋友，我接受不了，对方冷冷的说这跟你有什么关系
7月15号左右，我给对方发微信，对方抛过来一个冷冷的笑脸说，大兄弟，放手吧，那一刻心都碎了
谈恋爱这么多年对方常挂在嘴角边的一句话就是：你信不信我分分钟都能找到一个新男朋友
7月1号我送对方去车站坐车，对方很生气的说了一句，你信不信我打个电话就有人开车过来接我
7月5号左右，我精神开始不正常了，几乎发疯的状态：
做了一些比较极端的事，然后再打电话对方的态度就是恶狠狠的了：
你人品有问题
在你身上看不到未来看不到希望
你不要再那么幼稚那么天真了好不好，我想要的生活你给不了
精神上没有安全感，经济上也没有安全感，两样你总得给我一样吧
我男朋友特别宠我 什么都愿意听我的
2018年7月28号下午把微信拉黑，从此就是黑名单状态
2018年10月1号，微信群我就说了一句话，对方立马退群
你以后不要再给我和我家人打电话了，反正当初我们家人都看不上你
（7月份大概是婚纱照刚打印出来的时候）我现在还能接你电话，是我男朋友怕你伤心难过，哪天我男朋友不高兴了我立马把你拉黑，听到自己的未婚妻跟自己说这样的话，那一刻我真的有跳楼的冲动。
没有哪一个男朋友会希望自己的女朋友与前任有瓜葛
我就是终身不嫁我都不会嫁给你
自己拿个镜子照照，自己那么丑，还作怪
我给对方打电话哭着求对方回来吧，我知道错了，浪子回头金不换，修修补补还是好的，对方说我的东西坏了就是直接扔了，从来不修
你们家买的房子我一点都不满意
我问对方，这么狠心放下7年的感情吗，对方回答：我的日子还长着呢，以后还有很多个7年，我不能把自己的未来都葬送在你那里
8月中旬，我买了两份保险，一份人身意外保险，一份大病医疗保险，因为那时候真的就觉得自己的时日不多了，快要死掉的状态了，生怕自己那天想不开就死掉了，这样对自己的家人也有个交代
连续三个月没好好吃过饭，没睡过安稳觉，整天就是死撑着，实在撑不住了就闭上眼歇一会。
8月22号，身体终于坚持不住了，住院了，还动了小手术。很疼很疼。
2019年1月29号，一点左右到郑州，第二天早上7点多起来，特别冷那天，买了一束花过去，对方不见，答，是不是你觉得我有男朋友了你心里觉得过不去才不放过我的，你自己不好过，不也让我跟着不好过吗，你怎么那么自私，你要是看不过去，我现在就可以跟我男朋友分手，即使这样我也不会回头找你的，是你自己一手毁了你自己的感情，跟我有什么关系。
2019年2月2号。心够疼了，终于下定决心，告诉自己放手吧，寻找自己的未来，努力在北京打出一片天。当自己坚持不住的时候，就在草稿箱里写邮件，先不发送，有可能过两天就不会有想给对方发邮件的想法了。
